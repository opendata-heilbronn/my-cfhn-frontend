Speichert Zeitserien

InfluxDB ist eine Datenbank für Zeitserien, also Werte, die sich über einen Zeitraum verändern. Die BUGA-Solardaten werden hier gespeichert und über Grafana visualisiert. InfluxDBs bitte per Telegram bei patrick246 beantragen (siehe Button).
InfluxDB beantragen

Überwacht Dienste

Prometheus speichert Metriken von Diensten, die diese im Prometheus-Format anbieten. Nach der Erfassung können diese mit Queries weiterverarbeitet werden. Für Dienste, die diese nicht bereitstellen, gibt es oft sogenannte Exporter, die den Job übernehmen.

Neue zu überwachende Dienste können auf dem Code for Heilbronn Server mit Docker-Labels automatisch erkannt werden. Falls die Dienste außerhalb vom Server laufen, kann in Gitlab ein neues Ziel hinzugefügt werden.
